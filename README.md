# Enabling Voice AI Phone Number in Lowcoder

## Problem Statement

When a new agency phone number is created, it does not automatically appear in the **Lowcoder** tool. This requires manual configuration in the admin dashboard.

---

## Solution

### Step-by-Step Process

1. Navigate to **Admin Dashboard**
2. Go to **Agency Configs**
3. Select **Chatbot** section
4. Update the **Retell Call Number** field with the newly created agency phone number
5. Save the configuration

---

## Configuration Details

**Location:** Admin Dashboard > Agency Configs > Chatbot

**Field to Update:** Retell Call Number

**Action Required:** Manual entry of the agency phone number

---

## Verification

After updating the configuration:

1. Navigate to the **Lowcoder** tool
2. Verify that the phone number now appears in the available options
3. Test the phone number integration to ensure it's functioning correctly

---

## Important Notes

- This is a **manual configuration step** that must be performed for each new agency phone number
- The phone number will not sync automatically to Lowcoder without this update
- Ensure the phone number format matches the expected format in the Retell system
